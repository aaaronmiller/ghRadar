# Quickstart: Advanced GitHub Discovery & Agentic Intelligence

1. Open ghRadar and navigate to the Advanced Discovery view.
2. Choose a preset profile (e.g., "Top Agent Frameworks") or define filters:
   - Keywords in name/description/topics/README
   - Activity thresholds (recent commits, contributors)
   - Quality indicators (readme, license, tests, CI)
   - Popularity/community metrics
3. Run the search to see a ranked list with score breakdowns.
4. Save the profile to re-run later.
5. Click "Update" to refresh all saved profiles and store new snapshots.
6. Use the Trends and Comparison views to:
   - Inspect how repositories evolve over time
   - Compare candidates side-by-side
7. For agents/headless use, point them at the exported JSON views of:
   - Current ranked lists
   - Historical snapshots
   - Exemplar project sets
